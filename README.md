# 🧠 Atividades Java - Alura

Este repositório contém exercícios e projetos desenvolvidos durante os cursos da plataforma [Alura](https://www.alura.com.br/) voltados para o aprendizado da linguagem Java.

## 📚 Conteúdo

As atividades estão organizadas cobrindo conceitos fundamentais e intermediários de Java:

- ✅ Sintaxe básica
- 🔄 Estruturas de controle
- 🧮 Orientação a Objetos (POO)
- 📦 Pacotes e organização
- 🗃️ Manipulação de arquivos
- 🌐 Consumo de APIs

## 🚀 Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/ViniciusLCarvalho/Atividades-Java-Alura.git

2. Abra o projeto em sua IDE Java preferida (IntelliJ, Eclipse, VS Code).
3. Certifique-se de ter o JDK instalado (versão 17 ou superior recomendada).
4. Execute os arquivos .java individualmente conforme o exercício desejado.
